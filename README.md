# Adapter-common-lib

Библиотека, которая включает общие для всех адаптеров классы.


### Настройки

Добавить в `pom.xml` в зависимости

```
<dependency>
    <groupId>com.rbkmoney</groupId>
    <artifactId>adapter-common-lib</artifactId>
    <version>0.0.7-SNAPSHOT</version>
</dependency>
```
